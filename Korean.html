  • 제품 설명: https://cloud.toast.com/service/spellchecker

  • 주소 데이터를 검색 엔진용 컬렉션으로 덤프하기 위한 스크립트
  • 검색 엔진
  • 스프링 API
  • TOAST Cloud Console 제품 사용 화면, 제품 소개 및 샘플 화면

이밖에 업무를 위한 내부 위키 문서와 사용자를 위한 설명서를 작성하였고, 개발 이후는 계속 서비스 운영 업무를 맡았습니다.
